# Al--Resume-Analyzer
 AI resume Analyzer for freshers


## Problem Statement
Freshers do not know which technical skills are missing in their resume.

## Solution
This web application analyzes a resume PDF, finds skills, shows missing skills,
and calculates an ATS score.

## Tech Stack
HTML, CSS, JavaScript, AI tools

## AI Tools Used
Lovable, ChatGPT

## Setup Instructions
1. Open the deployed project link
2. Upload resume PDF
3. View skills and ATS score
